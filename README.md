# Text Processing - Varianta Secventiala

## Descriere
Acest proiect realizeaza numararea frecventei cuvintelor dintr-un fisier text folosind procesare secventiala.

## Algoritm
Programul:
1. Citeste fisierul text
2. Extrage cuvintele
3. Elimina semnele de punctuatie
4. Transforma literele in litere mici
5. Numara aparitiile cuvintelor
6. Afiseaza rezultatele

## Limbaj utilizat
C++

## Biblioteci utilizate
- iostream
- fstream
- string
- map
- chrono

## Specificatii sistem
- Procesor: Intel Core i5
- RAM: 8 GB
- Sistem de operare: Windows 11
